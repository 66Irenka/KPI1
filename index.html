Опис курсу

Курс «Проєктування WEB-застосувань» спрямований на формування практичних навичок веб-розробки. Протягом роботи над лабораторними студент опановує семантичну HTML-розмітку, каскадні таблиці стилів CSS, основи JavaScript та принципи створення клієнтських веб-інтерфейсів.

На основі методичних вказівок студенти послідовно створюють структурований веб-застосунок: формують таблиці, зображення, списки, форми, здійснюють стилізацію елементів, опрацьовують макет у Figma та переносять його у HTML/CSS. Під час виконання робіт також використовується Git і GitHub для ведення репозиторію та публікації результатів на GitHub Pages.

У підсумку студент отримує власний інтерактивний застосунок та вміє працювати з HTML, CSS, базовим JavaScript, створювати інтерфейси, стилізувати компоненти, підключати зовнішні ресурси та оформлювати професійні звіти у вигляді веб-сторінок.

Інструкції для користувачів

  • Оберіть лабораторну роботу у верхньому меню.
  • Для перегляду звіту скористайтеся лівим меню відповідної лабораторної.
  • Демонстрацію проєкту можна переглянути у вбудованому фреймі (iframe), якщо це передбачено.
  • Рекомендовано відкривати сторінки на локальному або віддаленому сервері.
