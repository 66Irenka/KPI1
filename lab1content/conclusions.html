Висновки до Лабораторної роботи №1

У ході виконання лабораторної роботи №1 було набуто практичних навичок роботи з системою контролю версій Git та хостингом GitHub. Зокрема, було виконано реєстрацію на GitHub, створено репозиторій для власного WEB-застосунку та звітного HTML-документа, а також здійснено клонування проєктів на локальний комп'ютер.

Було створено шаблон звітного HTML-документу для відображення результатів усіх лабораторних робіт, використовуючи наданий орієнтовний макет. Цей документ містить необхідну структуру навігації для відображення змісту лабораторних робіт.

Основні результати роботи:

  • Опис предметної галузі та бізнес-логіки майбутнього WEB-застосунку, а також формування функціональних і нефункціональних вимог до нього.
  • Застосування структурної та семантичної HTML-розмітки при створенні головної сторінки застосунку.
  • Успішне використання основних HTML-тегів та атрибутів для створення таблиць, зображень та списків (нумерованих та маркірованих) у звітному документі.
  • Набуття досвіду роботи з асинхронним завантаженням HTML-фрагментів (як видно з наданого коду `LB1.html`), що забезпечує динамічність відображення змісту звіту.

Таким чином, мету лабораторної роботи досягнуто, отримані знання та навички є фундаментальними для подальшої розробки та документування WEB-застосунків.
