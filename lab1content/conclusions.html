Відповіді на контрольні запитання

  1. Для чого використовується HTML?

    HTML використовується для структуризації та опису вмісту веб-сторінки. У моєму проєкті HTML формує каркас інтерактивного каталогу книг: шапку сайту (<header>) з назвою «Інтерактивний каталог», панеллю фільтрів і навігацією; основну частину з каталогом книг (<main>, <section>), блогову секцію, галерею, таблицю з рекомендованими книгами та футер з контактами і формою зворотного зв’язку.

  2. Які основні структурні елементи HTML-документа?

    Основні елементи HTML-документа:

    • <html> — кореневий елемент;
    • <head> — містить метадані, шрифти, стилі, заголовок сторінки;
    • <body> — уміщує видимий користувачу вміст.

    У моєму застосунку всередині <body> використано семантичні блоки:

    • <header> — шапка сайту з логотипом і фільтрами;
    • <main> — місце рендера каталогу книг;
    • <section> — блог, галерея, таблиця книг;
    • <footer> — контакти та форма зворотного зв’язку;
    • <div> — службові контейнерні блоки;
    • модальне вікно реалізовано через <div id="bookModal">.
  3. Чим відрізняються списки <ul> та <ol>?

    <ul> створює маркований список, у якому порядок елементів не має значення. <ol> — нумерований список, де елементи мають порядковий номер.

    У моєму проєкті <ul> використано в блозі та при перерахунку тематичних тегів, тоді як <ol> — у звітній документації (кроки, алгоритми та пояснення).

  4. Що таке атрибут id та class?

    Атрибут id задає унікальний ідентифікатор елемента. Він використовується для роботи JavaScript-скриптів, керування модальним вікном та для навігації через якірні посилання.

    У моєму застосунку приклади id:

    • id="searchInput" — поле пошуку;
    • id="sortSelect" — сортування книг;
    • id="home", id="about", id="contact" — якірна навігація;
    • id="bookModal" — модальне вікно книги.

    Атрибут class застосовується для групування елементів, які потрібно стилізувати однаково. Один клас може повторюватися безліч разів.

    Приклади class у моєму HTML:

    • app-header, app-bar, filters-card — елементи шапки;
    • blog-card — картки блогу;
    • gallery-image — зображення галереї;
    • top-books-table — таблиця рекомендованих книг;
    • footer-input, footer-button — елементи форми;
    • modal-link, modal-image — елементи модального вікна.
  5. Які теги застосовуються для створення таблиць?

    Для створення таблиць застосовуються такі елементи:

    • <table> — контейнер таблиці;
    • <thead> — заголовок таблиці;
    • <tbody> — основний вміст таблиці;
    • <tr> — рядок таблиці;
    • <th> — комірка заголовка;
    • <td> — комірка з даними.

    У моєму проєкті таблиця використовується в секції «Авторська добірка книг…», де відображається список із назвами книжок, авторами та жанрами. Вона стилізована через клас top-books-table.

Усі відповіді сформовані на основі практичного застосування HTML у моєму інтерактивному каталозі книг, що використовує Google Books API.
