Відповіді на контрольні запитання

  1. Для чого використовується HTML?
    HTML використовується для структуризації та опису вмісту веб-сторінки. У моєму проєкті HTML задає каркас інтерактивного каталогу: шапку (<header> з назвою «Інтерактивний каталог»), основну частину з каталогом книг (<main>, <section>), блог, галерею, футер з контактами та форму зворотного зв’язку.
  2. Які основні структурні елементи HTML-документа?
    Основними структурними елементами є: <html> – кореневий елемент документа, <head> – містить метадані (кодова сторінка, назва сторінки, підключення шрифтів та стилів), <body> – містить увесь видимий користувачу вміст. Усередині <body> у моєму застосунку додатково використовуються семантичні блоки <header>, <main>, <section>, <footer>, що логічно розділяють інтерфейс на шапку, основний контент, блог, галерею та блок контактів.
  3. Чим відрізняються списки ul та ol?
    Тег <ul> створює маркований (невпорядкований) список, де елементи позначаються однаковими маркерами без нумерації. Тег <ol> створює нумерований (упорядкований) список, де кожен елемент має порядковий номер. У моєму застосунку для навігації використано окремі посилання всередині <nav>, але для відображення переліків (наприклад, у звітному документі або в описах) доцільно застосовувати саме <ul> для маркованих списків та <ol> – для кроків або інструкцій.
  4. Що таке атрибут id та class?
    Атрибут id задає унікальний ідентифікатор елемента на сторінці. Його використовують для точкового звернення в JavaScript або CSS, а також для навігації за якірними посиланнями. Наприклад, id="home", id="about", id="contact" дозволяють переходити до відповідних секцій каталогу, а id="bookModal" – керувати модальним вікном з інформацією про книгу.
    Атрибут class використовується для об’єднання кількох елементів в групи для спільного стилізування. У моєму HTML класами оформлені майже всі блоки інтерфейсу: app-header, blog-card, gallery-image, top-books-table, footer-button тощо. Один клас може бути призначений багатьом елементам, а елемент може мати кілька класів.
  5. Які теги застосовуються для створення таблиць?
    Для створення таблиць у HTML використовуються такі теги: <table> – контейнер таблиці, <thead> – секція заголовка таблиці, <tbody> – основна частина таблиці, <tr> – рядок таблиці, <th> – комірка-заголовок, <td> – звичайна комірка з даними. У моєму застосунку ці елементи використано в блоці «Авторська добірка книг…» для відображення таблиці з книжками (назва, автор, жанр).

Ці відповіді демонструють розуміння основ HTML та їх застосування у моєму проєкті інтерактивного каталогу книг.
