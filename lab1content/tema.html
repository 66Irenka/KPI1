Тема та мета лабораторної роботи №1

Тема: Базова структура HTML-документа. Основні теги, списки, таблиці, форми та зображення.

Мета: ознайомитися з основними конструкціями HTML, навчитися будувати структуру веб-сторінок, використовувати базові елементи розмітки та організовувати контент.

У межах лабораторної роботи було створено веб-сторінку, що демонструє роботу основних елементів HTML, а також їх застосування у власному веб-застосунку — інтерактивному каталозі книг, який працює на основі Google Books API.

Репозиторій проєкту: https://github.com/66Irenka/IP-zp51_appWEB-PavlenkoIryna-FIOT-2025

Основні завдання, виконані у ході роботи:

Опис застосунку

У межах лабораторної роботи створено інтерактивний веб-каталог книг, який дозволяє виконувати пошук, фільтрацію, сортування та перегляд інформації з Google Books API.

Функціональні можливості:

  • пошук книг за назвою, автором або ISBN;
  • сортування за назвою, автором, роком видання;
  • перегляд детальної інформації у модальному вікні;
  • галерея найчастіше перегляданих обкладинок;
  • форма зворотного зв’язку у футері.

Структура застосунку включає: головну сторінку, блок фільтрів, каталог результатів, блогову секцію, галерею та футер.

Предметна область: веб-розробка, робота з API, інтерактивні інтерфейси.

Технології: HTML5, CSS3, JavaScript, Fetch API.

Демонстраційна версія застосунку доступна на GitHub Pages: https://66irenka.github.io/IP-zp51_appWEB-PavlenkoIryna-FIOT-2025/

HTML-код таблиці

У застосунку використано таблицю для відображення списку рекомендованих книг.

<table class="top-books-table">
  <tr>
    <th>#</th>
    <th>Назва</th>
    <th>Автор</th>
    <th>Жанр</th>
  </tr>
  <tr>
    <td>1</td>
    <td>1984</td>
    <td>Джордж Орвелл</td>
    <td>Антиутопія</td>
  </tr>
</table>

Таблиця містить заголовки стовпців та кілька рядків з інформацією про книги, включаючи їх назви, авторів та жанри.

HTML-код зображення

Зображення використано у галереї найчастіше перегляданих книг.

<img src="images/1.jpg" alt="Обкладинка книги" class="gallery-image">

Галерея обкладинок книг.
Рис. X – Галерея найчастіше перегляданих книг.

Кожне зображення має атрибути src (шлях до файлу зображення), alt (альтернативний текст для доступності) та клас для стилізації.

HTML-код форми

Форма зворотного зв’язку розміщена у футері.

<form class="feedback-form">
  <input type="email" placeholder="Ваш Email">
  <textarea placeholder="Ваш коментар"></textarea>
  <button type="submit">Надіслати</button>
</form>

Форма у футері.

Користувачі можуть залишити свій відгук, вказавши електронну пошту (необов’язково) та коментар.

async function loadBooks() {
  const url = "https://www.googleapis.com/books/v1/volumes?q=subject:fiction";
  const response = await fetch(url);
  const data = await response.json();
  renderBooks(data.items);
}

Лабораторна робота 1: Основи HTML та структура веб-документа

У ході виконання першої лабораторної роботи було детально опрацьовано базові принципи побудови HTML-документа, а також призначення й особливості використання основних структурних елементів мови розмітки.

  • Досліджено фундаментальну структуру HTML-файлу та призначення його ключових секцій: <html> – кореневий контейнер документа, <head> – область метаданих і службових налаштувань, <body> – простір візуального вмісту сторінки.
  • Опрацьовано використання основних елементів текстової розмітки: заголовків <h1>–<h6>, абзаців <p>, списків <ul>, <ol>, <li>, а також гіперпосилань <a>.
  • Вивчено механізм побудови таблиць: <table>, <thead>, <tbody>, <tr>, <th>, <td>.
  • Ознайомлено з можливостями застосування класів (class) та ідентифікаторів (id) для подальшої стилізації елементів за допомогою CSS.
  • Сформовано логічну й семантично обґрунтовану структуру веб-сторінки, що забезпечує коректне відображення, зручність навігації та придатність до подальшої розробки.

Отримані результати створюють необхідну теоретичну та практичну основу для подальшого вивчення веб-технологій, зокрема опанування CSS, JavaScript та побудови інтерактивних веб-застосунків.
