Висновки до лабораторної роботи №2

У ході виконання лабораторної роботи №2 було закріплено теоретичні знання та отримано практичні навички роботи з каскадними таблицями стилів (CSS) на прикладі власного веб-застосунку «Інтерактивний каталог книг».

  • Опрацьовано основні способи підключення стилів до HTML-документа та обґрунтовано доцільність використання зовнішнього CSS-файлу style.css для даного проєкту.
  • Застосовано селектори тегів, класів та ідентифікаторів для оформлення ключових елементів інтерфейсу: шапки, блоку фільтрів, секції блогу, галереї, таблиці рекомендацій та футера.
  • Продемонстровано використання вкладених, дочірніх, атрибутних та універсальних селекторів, а також псевдокласів для реалізації більш гнучкого та контекстного оформлення.
  • Виконано стильове оформлення текстових елементів, таблиць, списків, фонових блоків та елементів керування, що позитивно вплинуло на читабельність, зручність сприйняття інформації та візуальну цілісність інтерфейсу.
  • Здійснено аналіз ролі CSS у сучасній веб-розробці та визначено, що саме каскадні таблиці стилів забезпечують можливість відокремити логічну структуру документа від його презентаційної складової.

Отримані результати створюють необхідне підґрунтя для подальшого вивчення просунутих можливостей CSS (адаптивна верстка, анімації, фреймворки) та інтеграції стилів із JavaScript для розробки повноцінних інтерактивних веб-застосунків.
