Інші селектори

Окрім базових селекторів тегів, класів та ідентифікаторів, у проєкті «Інтерактивний каталог» використовуються й інші види селекторів: вкладені, дочірні, селектори атрибутів, псевдокласи та універсальний селектор.

1. Вкладені (дочірні) селектори

.blog-card h3 {
  margin-bottom: 8px;
}

.blog-card p {
  font-size: 0.95rem;
}

Селектори .blog-card h3 та .blog-card p застосовуються лише до заголовків і тексту всередині блоку з класом blog-card, не змінюючи інші заголовки на сторінці.

2. Селектор нащадків для галереї

.gallery-grid .gallery-item img {
  display: block;
}

Такий запис дозволяє застосувати правило тільки до зображень всередині галереї, не торкаючись інших <img> в інтерфейсі.

3. Селектор атрибута

input[type="email"] {
  font-style: italic;
}

Селектор input[type="email"] використовується для стилізації поля вводу електронної пошти у формі зворотного зв’язку у футері.

4. Псевдокласи (hover)

.main-nav a:hover {
  text-decoration: underline;
}

.footer-button:hover {
  cursor: pointer;
}

Псевдоклас :hover змінює вигляд посилань у меню при наведенні та додає інтерактивність кнопці надсилання форми.

5. Універсальний селектор

* {
  box-sizing: border-box;
}

Універсальний селектор * застосовується до всіх елементів сторінки. У цьому випадку він задає режим підрахунку розмірів із урахуванням полів і рамок, що спрощує адаптивну верстку.

6. Сусідні та комбіновані селектори

У цьому фрагменті CSS використано декілька «інших» селекторів:

  • Атрибутний селектор – для стилізації input[type="text"]
  • Сусідній селектор (+) – для стилізації елемента .blog-subtitle після h2
  • Комбінований сусідній селектор – для розміщення кнопки #resetSearchButton після поля вводу
