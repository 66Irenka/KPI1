Селектори класу

Селектори класу використовуються для стилізації групи елементів, які мають однакову роль в інтерфейсі. У проєкті «Інтерактивний каталог» класи активно застосовуються для оформлення шапки, карток блогу, галереї, таблиць та форм.

Приклад 1. Шапка застосунку

.app-header {
  padding: 20px 0;
}

.app-title-group {
  display: flex;
  flex-direction: column;
}

Клас app-header застосовується до елемента <header>, який містить назву каталогу та панель фільтрів. Клас app-title-group оформлює блок із заголовком і підзаголовком.

Приклад 2. Картки блогу

Клас blog-card використовується для оформлення кожної статті в секції «Наш блог» (заголовок, текст, тег). Клас blog-meta задає стиль для підпису з тегом статті (наприклад, «Самоосвіта», «Українська література», «IT & книги»).

Приклад 3. Галерея зображень

.gallery-image {
  width: 100%;
  display: block;
}

.gallery-item {
  border-radius: 8px;
}

Класи gallery-item та gallery-image використовуються у секції «Найчастіше переглядають» для оформлення карток із обкладинками книг. Завдяки ним зображення заповнюють блок рівномірно, а елементи галереї мають узгоджений вигляд.

Приклад 4. Таблиця рекомендованих книг

.top-books-table {
  width: 100%;
  border-collapse: collapse;
}

Клас top-books-table задає базовий вигляд таблиці з авторською добіркою книг: ширину, стикування меж, зовнішній вигляд шапки та рядків.
