CSS: шрифти, текст, таблиці, фон, контур, списки, просунуті властивості

1. Оформлення шрифтів та тексту

body {
  font-family: "Montserrat", sans-serif;
}




2. Оформлення таблиць






Ці правила формують охайну структуру таблиці з рекомендованими книгами,
забезпечують зручну для читання ширину та внутрішні відступи в комірках.




3. Фон і колір



.blog-section {
  /* приклад: фоновий колір або градієнт */
  /* background: #f7f7f7; */
}

.app-footer {
  /* темніший фон для відокремлення футера */
  /* background: #111; */
}




Фонові кольори та можливі градієнти використовуються для візуального
відокремлення секцій (блог, футер) та створення цілісного стилю застосунку.




4. Контур і рамки




  .blog-card {
  border-radius: 10px;
}

.gallery-item {
  border-radius: 8px;
}





Заокруглені кути на картках блогу та елементах галереї роблять інтерфейс
більш «м’яким» та сучасним.




5. Списки



У навігації використовується набір посилань, а в звітному документі – списки:



ul {
  /* базове оформлення маркованих списків у звіті */
}




Списки дозволяють структуровано подавати інформацію про функції застосунку,
етапи роботи, висновки та відповіді на контрольні питання.




7. Кнопки 



    На прикладі кнопки у футері видно використання класу .footer-button для оформлення
    інтерактивного елемента форми: відступи, шрифт, округлення кутів, тінь, а також плавні переходи
    при наведенні.






6. CSS “просунутий” (інтерактивність та ефекти)



.scroll-top-btn {
  /* приклад фіксованого позиціонування та плавної появи */
  /* position: fixed; bottom: 20px; right: 20px; */
}






У проєкті застосовуються сучасні прийоми: фіксовані кнопки (scroll-to-top),
модальні вікна, стилізація елементів форм, використання псевдокласів
для інтерактивних станів (hover, focus тощо).
